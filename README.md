# ⚔️ Mork's Quest

An atmospheric, text-based RPG engine built in Python. Navigate a dangerous forest, delve into a damp cave, and reclaim the legendary Gemstone from **Steve the Troll**.

## 🎮 Features
- **3 Unique Classes**: Play as a high-HP **Warrior**, a tactical **Mage**, or a high-damage **Rogue**.
- **Dynamic Combat**: Features unique monster attacks, crit logic, and mana management.
- **Inventory System**: Find and use items like the **Shadow Dagger** and **Mana Potions**.
- **Visual UI**: Dynamic, color-coded health and mana bars using Colorama.

## 🛠️ Installation & Setup

To play Mork's Quest, you need to have **Python** installed.
